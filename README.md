# Todosha
Simple toDo

  Возможности:
- Хранит список задач(+добавление, удаление, редактирование)
- Подсветка просроченных и ближайших тасков
- Отправляет push по выбранной дате
- Удаление с листа(SwipeToDismiss)
- Поиск по списку

Стэк:
- BroadcastReceiver
- SwipeToDismiss
- JSON

Комментарий: не протестировано
